You see the comma ( , ) because you are adding the whole array to textContent.

Look at this line:

Greeting_tag1.textContent = Greeting + " "

Here Greeting is an array:

["welcome back", "suraj", "this website help to shop B2B"]

When JavaScript converts an array to a string, it automatically joins items with commas.

So it becomes:

welcome back,suraj,this website help to shop B2B

That is why you see ,.
